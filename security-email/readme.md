### 注册
知识点复习：

    JPA相关
    
流程：

    首页--》注册--》填写基本信息--》提交表单--》
    持久化基本信息到数据库、发送验证邮件--》
    邮箱验证--》登录页登录

需求：

    1、所有必填字段是否都已填写（无空白字段或 null 字段）
    2、电子邮件地址是否有效（格式是否正确）
    3、密码确认字段与密码字段是否匹配（前端匹配，后端不需要）
    4、帐户是否已经注册
    5、为每个用户生成校验Token，并持久化
    6、给注册邮箱发送带有校验Token的邮件
    7、密码强度校验提示
    8、注册登录都使用HMAC 算法校验
